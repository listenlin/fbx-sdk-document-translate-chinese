## 什么是FBX SDK？
Autodesk FBX SDK技术包含了一整套工具，用来让开发者对3D模型进行快捷的导入、修改和导出等操作，而FBX SDK是整套工具的一部分。通过Autodesk FBX来开发混合2D和3D数字内容的应用程序，有助于许多组织团队更有效的参与到数字媒体娱乐行业中来。
